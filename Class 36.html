If-Else Statements


JavaScript If-else: The JavaScript if-else statement is used to execute the code whether condition is true or false. There are two forms of if statement in JavaScript. 1. If Statement 2. If else Statement


IF evaluates the content only if expression/condition is true. The signature of JavaScript if statement is given below. If(condition){
.. code to be executed when it is true ..
}


IF-ELSE evaluates the content whether condition is true of false. The syntax of JavaScript if-else statement is given below. if(expression){
//content to be evaluated if condition is true
}
else{
//content to be evaluated if condition is false
}
